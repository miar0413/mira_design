# Projects

## ByteHi Customer Service Upgrade

客户平台是TikTok统一的用户反馈处理平台，是提升用户体验的重要工具之一。

### Overview

- **Role**: Lead Product Designer
- **Timeline**: 2023.01 - 2023.06
- **Platform**: Web Application

### Challenge

提升客服效率和用户满意度是平台面临的主要挑战。

### Solution

通过系统化的设计方法，我们：

1. 重新设计了工单处理流程
2. 优化了界面交互
3. 引入了智能辅助功能

### Impact

- 客服效率提升 40%
- 用户满意度提升 25%
- 平均响应时间减少 50%
